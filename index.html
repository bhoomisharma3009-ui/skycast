What this integration demonstrates

  

    

      
🔐

      
API Key Auth

      
Every request requires a valid x-api-key header. Returns 401 without it.

    

    

      
⚡

      
In-Memory Cache

      
Responses cached for 10 minutes. Repeat requests return instantly, saving upstream calls.

    

    

      
🛡

      
Input Validation

      
Joi schemas validate all query params. Missing or invalid inputs return structured 400 errors.

    

    

      
🔄

      
Retry Logic

      
Axios interceptor retries on 5xx errors with exponential backoff (up to 2 attempts).

    

    

      
🚦

      
Rate Limiting

      
60 requests/minute per client. Protects both the service and the upstream API quota.

    

    

      
📋

      
Clean Error Handling

      
All errors return consistent JSON with status codes — no stack traces leaking to clients.
